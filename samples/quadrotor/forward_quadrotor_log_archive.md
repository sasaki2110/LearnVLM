## Run 6-10 — 速度Dの実装と符号修正の試行錯誤

### Run 6 — 速度D有効（kd_x=1.0, kd_y=1.0）

- **結果**: 不安定になり、墜落寸前で縦向きに回転
- **問題**: 速度Dのゲイン（kd_x=1.0, kd_y=1.0）が強すぎる

### Run 7 — 速度D有効（kd_x=0.1, kd_y=0.1）

- **結果**: 安定性は改善したが、速度Dが速度を抑制せず、方向を変えているだけ
- **問題**: ゲインが弱すぎる可能性

### Run 8 — 速度D有効（kd_x=0.1, kd_y=0.1）、チョン中は速度D無効化

- **結果**: チョン中の速度D無効化は正しく動作したが、チョン終了後の速度抑制が不十分
- **問題**: 速度Dの符号が逆になっている可能性

### Run 9 — 速度D有効、符号修正

- **結果**: 符号修正によりvel_yはやや改善したが、vel_xは逆方向に加速
- **問題**: 符号を元に戻す必要がある可能性

### Run 10 — 速度D有効、符号を元に戻す

- **結果**: Run 8と完全に同じ結果
- **問題**: 符号の問題ではなく、ゲインが弱すぎる可能性

## Run 11-19 — 速度Dゲインの調整

### Run 11 — 速度D有効（kd_x=0.3, kd_y=0.1）

- **結果**: KD_Xを0.3に増やしたことで、vel_xの抑制が改善（約42%改善）
- **問題**: KD_Y=0.1ではvel_yの抑制が不十分

### Run 12 — 位置Pゲイン変更（KP_Y=0.3、ただしTARGET_Y=Noneのため無効）

- **結果**: Run 11と完全に同じ結果
- **問題**: TARGET_Y=Noneのため位置Pは無効化されている

### Run 13 — 速度D有効（kd_x=0.3, kd_y=0.3）

- **結果**: KD_Yを0.3に増やしたことで、Δyは大幅に改善したが、vel_yが異常に大きくなり、姿勢も大きく崩れている
- **問題**: KD_Y=0.3は強すぎる可能性

### Run 14 — 速度D有効（kd_x=0.3, kd_y=0.2）

- **結果**: KD_Yを0.2に下げたことで、Run 13より改善したが、vel_yが323.07 cm/sと大きい
- **問題**: Y方向の符号が逆になっている可能性が高い

### Run 15 — 速度D有効、Y方向の符号を修正

- **結果**: Y方向の符号を修正した結果、大幅に悪化
- **問題**: 元の符号（-kd_y * vel_y）が正しかった可能性が高い

### Run 16 — 速度D有効、Y方向の符号を元に戻す

- **結果**: Run 14と完全に同じ結果
- **問題**: 元の符号が正しいことが確認された

### Run 17 — 速度D有効（kd_x=0.3, kd_y=0.15）

- **結果**: KD_Yを0.15に下げたことで、vel_yが改善したが、Δyが悪化
- **問題**: トレードオフがある

### Run 18 — 速度D有効（kd_x=0.3, kd_y=0.175）

- **結果**: 中間値として機能し、Run 16とRun 17の中間の結果
- **問題**: Run 16（kd_y=0.2）の方がΔyが改善している

### Run 19 — 速度D有効（kd_x=0.3, kd_y=0.225）

- **結果**: KD_Y=0.225に設定した結果、Δyが-167.20 cmと改善したが、vel_yが355.30 cm/sと悪化
- **問題**: どちらを優先するかによる

## Run 20-22 — 位置Pの追加とゲイン調整、符号修正

### Run 20 — 位置P+速度D有効（kp_x=0.1, kd_x=0.3, kp_y=0.1, kd_y=0.225）、目標位置(0,0)

- **結果**: チョン中にも位置Pが動作し、問題が発生（高度が低下、姿勢が崩壊）

### Run 21 — 位置P+速度D有効、チョン中は位置P無効化

- **結果**: チョン中の位置P無効化は正しく動作したが、チョン終了後に姿勢が大きく崩れ、高度が低下
- **問題**: 位置Pのゲイン（kp_x=0.1, kp_y=0.1）が強すぎる可能性

### Run 22 — 位置P+速度D有効（kp_x=0.05, kd_x=0.3, kp_y=0.05, kd_y=0.225）、チョン中は位置P無効化

- **結果**: 位置Pのゲインを0.05に下げたことで、Δy、高度、姿勢が改善したが、vel_yが652.99 cm/sと異常に大きい

**次の対応事項**: 外側ループのゲインをさらに調整するか、内側ループと外側ループのバランスを再検討する必要がある。
